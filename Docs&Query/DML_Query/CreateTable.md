### **CREATE**: create a table
* `CREATE TABLE` table_name `(` <br />
   `column1` `datatype`, <br />
   `column2` `datatype`, <br />
   `column3` `datatype`, <br />
   `column4` `datatype`, <br />
   `);`
  - eg: <br />
    create table student (`id int not null`, <br />
    `name varchar(20)`, <br />
    `roll int`, <br />
    `primary key(id)`);
